# Food Shopping Cart

## Description

Food Shopping Cart is a web application developed using React.js. It allows users to view food items, add or remove products from the shopping cart, and calculate the total price dynamically. The project demonstrates React components, state management, and responsive UI design.

## Features

- Display food products
- Add items to the cart
- Remove items from the cart
- Real-time total price calculation
- Responsive user interface

## Technologies Used

- React.js
- JavaScript
- HTML
- CSS

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/saniyaahamad/FoodShoppingCart.git
   ```

2. Navigate to the project folder:
   ```bash
   cd FoodShoppingCart
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Run the application:
   ```bash
   npm start
   ```

5. Open your browser and visit:
   ```
   http://localhost:3000
   ```
